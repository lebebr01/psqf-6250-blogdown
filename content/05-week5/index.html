Introduction

This module will give some tools to use when performing exploratory data analysis, a framework that helps you to better understand your data prior to doing any more formal analysis.

Objectives

After completing this module, students will be able to:

  • Evaluate exploratory figures
  • Define exploratory data analysis
  • Interpret exploratory data analysis output
  • Create exploratory data analysis code
